# TDD Exercises

A collection of Test Driven Development Exercises designed to practice and improve your skills in **Test-Driven Development (TDD)** and code refactoring.
Exercise: https://www.theodinproject.com/lessons/node-path-javascript-testing-practice

### Installation

Clone the repository:

```bash
git clone https://github.com/eekoka/tdd-exercises.git
cd tdd-exercises
```

Install dependencies (if applicable):

```bash
# Example for Node.js
npm install
```

Run tests:

```bash
npm test
```
